markdown语法大全

一、标题

Markdown 支持两种标题的语法，类 Atx 和类 Setext 形式。

1.1 Atx形式的标题可以使用1~6个#分别代表一到六级标题

示例：

# 一级标题 #
## 二级标题 ##
### 三级标题 ###
#### 四级标题 ####
##### 五级标题 #####
###### 六级标题 ######

显示效果：

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

1.2 Setext 形式是用底线的形式，利用 = （最高阶标题）和 - （第二阶标题）（任何数量）

示例：

This is an H1
\=============

This is an H2
-————

显示效果：

This is an H1

This is an H2

二、斜体用*号包围，强调用*号包围,强调斜体用**包围

示例：

*斜体*
**强调
***强调斜体*

显示效果：

斜体
强调
强调斜体

三、在引用的文字前加>即可。引用也可以嵌套，如加两个>>三个>>>四个’’’

示例：

>这是引用的内容
>>这是引用的内容
>>>>>>>>>>这是引用的内容

显示效果：

这是引用的内容

这是引用的内容

这是引用的内容

四、分割线

三个或三个以上的 - 或者 * 都可以
示例：

-–
-—
*
***

显示效果：

五、列表

无序列表用 - + * 任何一种都可以有序则使用数字加点开头

5.1无序列表

无序列表用 - + *

示例：

- 列表内容
+ 列表内容
* 列表内容

显示效果：

  • 列表内容
  • 列表内容
  • 列表内容

5.2有序列表

数字加点

1. 列表内容
2. 列表内容
3. 列表内容

显示效果

  1. 列表内容
  2. 列表内容
  3. 列表内容

5.3 列表嵌套

上衣级与下一级之间敲三个空格即可

示例：
/+ 一级无序列表内容

   /+ 二级无序列表内容
   /- 二级无序列表内容
   /* 二级无序列表内容

/- 一级无序列表内容
/* 一级无序列表

/1. 一级有序列表内容

   /1. 二级无序列表内容
   /2. 二级无序列表内容
   /3. 二级无序列表内容

/2. 一级有序列表内容

显示效果：

  • 一级无序列表内容
    • 二级无序列表内容
      • 二级无序列表内容
      • 二级无序列表内容
  • 一级无序列表内容
    • 二级无序列表内容
    • 二级无序列表内容
    • 二级无序列表内容
  • 一级无序列表
  1. 一级有序列表内容
    1. 二级无序列表内容
    2. 二级无序列表内容
    3. 二级无序列表内容
  2. 一级有序列表内容

六、表格

示例：

head one head two head three
